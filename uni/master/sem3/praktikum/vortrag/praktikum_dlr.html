DLR - Allgemein

  • Das DLR ist das nationale Forschungszentrum der Bundesrepublik Deutschland für Luft- und Raumfahrt.
  • Die Forschungs- und Entwicklungsarbeiten sind in nationale und internationale Kooperationen eingebunden.
  • Das DLR ist für die Planung und Umsetzung der deutschen Raumfahrtaktivitäten sowie für die internationale Interessenswahrnehmung zuständig.

Versionsmanagement

  • Versionsmanagement ermöglicht das Arbeiten im Team.
  • Mehrere Mitarbeiter arbeiten gleichzeitig an den gleichen Dateien.
  • Die Versionskontrolle verhindert Chaos und Speicherplatzprobleme.
  • Durch update, edit, commit wird der Code aktuell gehalten
  • früher: CVS, heute: SVN
  • Mit Tags und Branches kann man Versionen "einfrieren" und auch später noch verfügbar machen.

  • Es muss eine sog. Commit-Datenbank angelegt werden.

ViewVC - Commitdatabase (cont.)

  • Changelog on-the-fly

ViewVC - Maintenance (CVS)

  • Die Commit-Datenbank sollte pro Commit aktualisiert werden.
  • Die Versionskontrollsysteme bieten hierzu sog. Hookskripte an.
  • Bei CVS-Projekten bietet sich das Hookskript loginfo an.
    • loginfo wird nach einem erfolgreichen Commit aufgerufen und bekommt eine Revisionsnummer und den Namen des Repositories.
    • Mit diesen Parametern wird der sog. loginfo-handler aufgerufen, welcher die Datenbank aktualisiert.

  • #viewvc loginfohandler
ALL /work/hinz_ja/install/viewvc/bin/loginfo-handler %{sVv}

ViewVC - Maintenance (SVN)

  • Bei SVN-Projekten wird das Skript post-commit benutzt.
    • post-commit wird nach einem erfolgreichen Commit aufgerufen und bekommt ebenfalls die nötigen Parameter vom SVN-System übergeben.
    • post-commit ruft dann den svndbadmin auf, welcher die Datenbank aktualisiert.

ViewVC - Maintenance (post-commit hook)

  • Das Hookscript post-commit aktualisiert die Commitdatenbank. 
    #!/bin/sh
# Jan Hinzmann 2006.04.26
# This script enables realtime updates to the commit database 
# for the viewvc query interface.

VIEWVC="/work/hinz_ja/install/viewvc"
REPOSITORY="$1"
REVISION="$2"

$VIEWVC/bin/svndbadmin rebuild "$REPOSITORY" "$REVISION"
